## Project Description
This project is a C# command-line cybersecurity awareness chatbot. The chatbot allows users to interact with the system and receive responses related to online safety practices and cybersecurity awareness.

## Features
- Greets the user
- Accepts user input
- Responds to cybersecurity-related questions
- Simulates a chatbot-style conversation

## Technologies Used
- C#
- .NET
- Visual Studio

## How the Program Works
The program starts by displaying a greeting message. It then waits for user input. Based on the user’s input, the program processes the request and provides a relevant cybersecurity response. The chatbot continues interacting until the user chooses to exit.

## How to Run the Program
1. Open the project in Visual Studio.
2. Open the solution file (.sln).
3. Build the solution.
4. Run the program.
5. Follow the on-screen instructions to interact with the chatbot.
